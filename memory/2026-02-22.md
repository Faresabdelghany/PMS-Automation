## Key Lessons Learned
- Sub-agent outputs MUST write to files — auto-announced results get lost when busy
- Telegram has 4096 char limit — sub-agent reports too long, need summaries
- Claude Code on Windows: use .cmd files for prompts, `--strict-mcp-config --mcp-config empty-mcp.json` to skip slow MCPs
- Claude Code gets rate limited when shared token with OpenClaw — avoid rapid successive spawns
- Break big Claude Code tasks into smaller pieces
